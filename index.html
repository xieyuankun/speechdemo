Quantized Prosody Learning in Rhythmic Speech Synthesis

Authors: Yutian Wang1, Yuankun Xie1, Huaichang Du2, Hui Wang3*, Qin Zhang3

Abstract: In recent years, text-to-speech (TTS) has made great strides with the application of deep learning. However, one of the shortcomings of the TTS model is that generating rhythmic speech is still very difficult. Furthermore, since prosody is affected by multiple speech characteristics, it is a more difficult problem to manually control the prosody of generated speech. In this paper, we propose a novel TTS architecture which can simulate the speaking prosody from reference audio and manually control the prosody by modifying the latent quantized vectors. Our system consists of three components: (1) Prosody encoder based on vector quantization method to learn the discrete latent representation from reference audio. (2) A sequence-tosequence speech synthesis network based on Tacotron that generates Mel spectrum from input text which is conditioned on the latent quantized vectors from prosody encoder. (3) A vocoder based on a modified WaveRNN model, which can get the final raw audio waves from the Mel spectrum. Experiments show that the speaking style of generated results is very close to the reference audios and the style can be controlled by manually changing the value of the latent quantized vectors.
